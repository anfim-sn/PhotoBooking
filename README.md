## Docker startup

1. Clone repo
2. Build docker `docker-compose -f docker-compose.build.yml build --no-cache`
3. Startup docker `docker-compose up -d`

## Stack

- React with Typescript
- C# with ASP.NET Core, InMemory DB, Swagger
- Docker (with docker-compose), Nginx
- Playwright
